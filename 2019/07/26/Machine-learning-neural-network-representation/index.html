[зӯҶиЁҳ] ж©ҹеҷЁеӯёзҝ’ зҘһз¶“з¶Іи·Ҝ ( Neural Network ) - иЎЁеұӨзөҗж§Ӣ

зӮәд»ҖйәјдҪҝз”ЁзҘһз¶“з¶Іи·Ҝ

  • еңЁзү№еҫөйҮҸеӨҡзҡ„жғ…жіҒдёӢпјҢеңЁйқһз·ҡжҖ§еӣһжӯёз®—жі•иЈЎпјҢжңғз”ўз”ҹеҫҲеӨҡзҡ„зү№еҫөй …пјҢйҖҷдёҰдёҚжҳҜдёҖеҖӢи§ЈжұәиӨҮйӣңйқһз·ҡжҖ§е•ҸйЎҢзҡ„еҘҪиҫҰжі•

  • еҒҮиЁӯжҲ‘еҖ‘иҰҒеҲҶйЎһ 50 * 50 еғҸзҙ зҡ„зҒ°йҡҺең–зүҮ :

    • жӯӨзү№еҫөйҮҸ n = 2500 ( RBG зҡ„и©ұе°ұжҳҜ n = 7500 )
    • йӮЈйәј xбөў * xвұј зҡ„жүҖжңүжўқд»¶е°ұжңү 2500 * 2500 / 2 вүҲ 300 иҗ¬
    • йҖҷжЁЈзҡ„иЁҲз®—жҲҗжң¬еӨӘй«ҳдәҶ

  • иҖҢзҘһз¶“з¶Іи·ҜжҳҜеҖӢиғҪиј•й¬Ҷи§Јжұәйқһз·ҡжҖ§е•ҸйЎҢзҡ„з®—жі•

  • зҘһз¶“з¶Іи·ҜжңҖеҲқз”ўз”ҹзҡ„зӣ®зҡ„жҳҜжЁЎж“¬дәәйЎһзҡ„еӨ§и…ҰпјҢж—©жңҹдёҚзӣӣиЎҢзҡ„еҺҹеӣ жҳҜпјҢз•¶жҷӮзЎ¬й«”з„Ўжі•ж”Ҝж’җеҰӮжӯӨйҫҗеӨ§зҡ„йҒӢз®—

зҘһз¶“з¶Іи·ҜжЁЎеһӢзҡ„йҒӢдҪң

зҘһз¶“з¶Іи·Ҝзҡ„жЁЎеһӢ

  • еңЁеүҚйқўијёе…Ҙзү№еҫө xвӮҖ ~ xвӮҷ
  • xвӮҖ жҳҜдёҖеҖӢеҒҸзҪ®е–®е…ғ ( bias unit )пјҢе§ӢзөӮзӮә 1
  • жңҖеҫҢзҡ„ијёеҮәеүҮжҳҜжҲ‘еҖ‘еҒҮиЁӯеҮҪж•ёзҡ„зөҗжһң
  • еңЁзҘһз¶“з¶Іи·ҜдёӯжңҖеҹәжң¬жҲ‘еҖ‘дҪҝз”Ёи·ҹеҲҶйЎһдёӯзӣёеҗҢзҡ„йӮҸијҜеҮҪж•ё пјҢжңүжҷӮжңғзЁұзӮәжҝҖеӢөеҮҪж•ё ( sigmoid activation function )пјҢд№ӢеҫҢд№ҹжңғжңүе…¶д»–зЁ®зҡ„еҮҪж•ёпјҢдҫӢеҰӮ : TanHгҖҒReLU
  • theta жңүжҷӮд№ҹиў«зЁұзӮәж¬ҠйҮҚ ( weights )

  • Layer 1 зӮә input layer
  • Layer 2 зӮә hidden layerпјҢеңЁ input layer е’Ң output layer д№Ӣй–“зҡ„йғҪжҳҜ
  • Layer 3 зӮә output layer

aбөўвҒҪКІвҒҫ зӮәеңЁ Layer j дёӯзҡ„ activation unit i
ОҳвҒҪКІвҒҫ зӮәжҺ§еҲ¶ Layer j еҲ° Layer j + 1 зҡ„ж¬ҠйҮҚзҹ©йҷЈ

  • ОҳвҒҪКІвҒҫ зҡ„з¶ӯеәҰ ( dimension ) зӮә sвұјвӮҠвӮҒ * (sвұј + 1)
  • sвұј зӮәеңЁ Layer j дёӯзҡ„ unit еҖӢж•ё

зҘһз¶“з¶Іи·Ҝзҡ„иЁҲз®—йҒҺзЁӢ

  • еҒҮиЁӯ x = aвҒҪВ№вҒҫ
  • aвҒҪКІвҒҫ = g(zвҒҪКІвҒҫ)пјҢдҫӢеҰӮ : aвҒҪВІвҒҫ = g(zвҒҪВІвҒҫ)
  • zвҒҪКІвҒҫ = ОҳвҒҪКІвҒ»В№вҒҫaвҒҪКІвҒ»В№вҒҫпјҢдҫӢеҰӮ : zвҒҪВІвҒҫ = ОҳвҒҪВ№вҒҫaвҒҪВ№вҒҫ
  • hОҳ(x) = aвҒҪКІвҒҫ = g(zвҒҪКІвҒҫ)пјҢжӯӨ Layer j зӮә output layer

зҘһз¶“з¶Іи·Ҝзҡ„жҮүз”Ё

йҒӢз®— OR

  • hОҳ(x) = g(-10 + 20xвӮҒ + 20xвӮӮ)
  1. xвӮҒ = 0 and xвӮӮ = 0 then g(вҲ’10) вүҲ 0
  2. xвӮҒ = 0 and xвӮӮ = 1 then g(10) вүҲ 1
  3. xвӮҒ = 1 and xвӮӮ = 0 then g(10) вүҲ 1
  4. xвӮҒ = 1 and xвӮӮ = 1 then g(30) вүҲ 1
  • g() зӮәйӮҸијҜеҮҪж•ё ( sigmoid function )

йҖІйҡҺйҒӢз®— XNOR

  • AND : ОҳвҒҪВ№вҒҫ = [-30 20 20]
  • NOR : ОҳвҒҪВ№вҒҫ = [10 -20 -20]
  • OR : ОҳвҒҪВ№вҒҫ = [-10 20 20]

е°ҮдёүеҖӢзөҗеҗҲеҫҢ :

  1. aвҒҪВІвҒҫ = g(ОҳвҒҪВ№вҒҫx)
  2. aвҒҪВівҒҫ = g(ОҳвҒҪВІвҒҫaвҒҪВІвҒҫ)
  3. hОҳ(x) = aвҒҪВівҒҫ

еӨҡйЎһеҲҘеҲҶйЎһйҒӢз®—

  • еңЁеӨҡйЎһеҲҘеҲҶйЎһжҷӮпјҢдёҰдёҚжҳҜ y = 1гҖҒy = 2гҖҒy = 3гҖҒy = 4вҖҰйҖҷжЁЈеҲҶйЎһ
  • иҖҢжҳҜжңүеӨҡеҖӢијёеҮәдҫҶеҖӢеҲҘиЎЁзӨәжҳҜеҗҰзӮәжӯӨеҲҶйЎһ

  • еҒҮиЁӯйҖҷж¬Ўжңү 4 зЁ®еҲҶйЎһпјҢйӮЈијёеҮәзҡ„жүҖжңүзЁ®йЎһеҰӮдёҠ

  • зҘһз¶“з¶Іи·Ҝзҡ„жЁЎеһӢеҸҜд»ҘйҖҷжЁЈиЁӯзҪ®
  • еңЁ Octave дёҠзҡ„еӨҡйЎһеҲҘеҲҶйЎһеҷЁеҮҪејҸ
  • еңЁ Octave дёҠзҡ„й җжё¬иЁҲз®—еҮҪејҸ
  • еңЁ Octave дёҠзҡ„й җжё¬иЁҲз®—еҮҪејҸ ( еӨҡеұӨзҘһз¶“з¶Іи·Ҝ )
